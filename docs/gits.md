# Usefull?

## [Branches](https://www.atlassian.com/git/tutorials/using-branches)

### List branches
```
git branch
```

### Create branch
```
git branch [name]

```

### Publish branch
```
git push -u origin [name]
```

### Delete branch
```
git branch -d [name]
```
or force delete
```
git branch -D [name]
```

### Rename branch to [name]
```
git branch -m [name]
```

### Checkout branch
```
git checkout [name]
```

### Create and check out 
```
git checkout -b [name]
```
